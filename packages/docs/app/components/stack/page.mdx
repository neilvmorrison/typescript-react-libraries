# Stack

A layout component that stacks its children vertically.

## Usage

```typescript
import { Stack } from '@libraries/component-library';

export function Example() {
  return (
    <Stack>
      [Example content]
    </Stack>
  );
}
```

## API Reference

### Import

```typescript
import { Stack } from '@libraries/component-library';
```

### Props

| Name       | Type                      | Required | Description              | Default |
| ---------- | ------------------------- | -------- | ------------------------ | ------- |
| `gap`      | `string                   | No       | The gap between children | "8px"   |
| `children` | `React.ReactNode`         | No       | Component content        | N/A     |
| `...props` | `[HTMLElementAttributes]` | No       | Standard HTML attributes | N/A     |

## Examples

### Basic

```typescript
<Stack>
```
