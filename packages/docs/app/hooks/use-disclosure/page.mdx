# useDisclosure

Control open/close state for modals, drawers, and similar components.

## Usage

```typescript
import { useDisclosure } from '@libraries/hooks';

export function ModalExample() {
  const { isOpen, open, close, toggle } = useDisclosure(false);

  return (
    <>
      <button onClick={open}>Open Modal</button>
      {isOpen && (
        <div className="modal">
          <h2>Modal Title</h2>
          <button onClick={close}>Close</button>
        </div>
      )}
    </>
  );
}
```

## API Reference

### Import

```typescript
import { useDisclosure } from '@libraries/hooks';
```

### Parameters

| Name           | Type      | Required | Description           | Default |
| -------------- | --------- | -------- | --------------------- | ------- |
| `defaultValue` | `boolean` | No       | Initial boolean value | `false` |

### Returns

| Name     | Type         | Description                              |
| -------- | ------------ | ---------------------------------------- |
| `value`  | `boolean`    | Boolean indicating if disclosure is open |
| `open`   | `() => void` | Function to open the disclosure          |
| `close`  | `() => void` | Function to close the disclosure         |
| `toggle` | `() => void` | Function to toggle the disclosure        |

### External Dependencies

None
