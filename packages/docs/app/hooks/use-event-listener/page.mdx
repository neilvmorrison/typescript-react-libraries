# useEventListener

Safely attach and remove event listeners with automatic cleanup.

## Usage

```typescript
import { useEventListener } from '@libraries/hooks';

export function ResizeListener() {
  useEventListener('resize', () => {
    console.log('Window resized');
  });

  return <div>Check the console on window resize</div>;
}
```

## API Reference

### Import

```typescript
import { useEventListener } from '@libraries/hooks';
```

### Parameters

| Name       | Type                            | Required | Description                          | Default |
| ---------- | ------------------------------- | -------- | ------------------------------------ | ------- |
| `event`    | `K extends keyof IEventMap`     | Yes      | Event name to listen for             | N/A     |
| `callback` | `(event: IEventMap[K]) => void` | Yes      | Function to execute when event fires | N/A     |

### Returns

| Name   | Type   | Description                                         |
| ------ | ------ | --------------------------------------------------- |
| (void) | `void` | No return value; automatically cleans up on unmount |

### Features

- Automatic cleanup on component unmount
- Supports both window and element events
- Type-safe event handling with full event map coverage
