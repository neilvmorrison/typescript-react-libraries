# useLocalStorage

Sync component state with browser localStorage.

## Usage

```typescript
import { useLocalStorage } from '@libraries/hooks';

export function Counter() {
  const { value: count, setValue: setCount } = useLocalStorage('count', 0);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  );
}
```

## API Reference

### Import

```typescript
import { useLocalStorage } from '@libraries/hooks';
```

### Parameters

| Name           | Type     | Required | Description                                   | Default |
| -------------- | -------- | -------- | --------------------------------------------- | ------- |
| `key`          | `string` | Yes      | localStorage key to use                       | N/A     |
| `initialValue` | `T`      | Yes      | Initial value if key doesn't exist in storage | N/A     |

### Returns

| Name          | Type                                        | Description                                                      |
| ------------- | ------------------------------------------- | ---------------------------------------------------------------- |
| `value`       | `T`                                         | Current stored value                                             |
| `setValue`    | `(value: T \| ((current: T) => T)) => void` | Function to update the stored value                              |
| `removeValue` | `() => void`                                | Function to remove value from storage and reset to initial value |
