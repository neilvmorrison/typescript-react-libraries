# useToggle

Simple boolean state toggle with optional initial value.

## Usage

```typescript
import { useToggle } from '@libraries/hooks';

export function Accordion() {
  const [isExpanded, toggle] = useToggle(false);

  return (
    <div>
      <button onClick={toggle}>
        {isExpanded ? 'Collapse' : 'Expand'}
      </button>
      {isExpanded && <p>Hidden content</p>}
    </div>
  );
}
```

## API Reference

### Import

```typescript
import { useToggle } from '@libraries/hooks';
```

### Parameters

| Name           | Type                  | Required | Description                                                         | Default |
| -------------- | --------------------- | -------- | ------------------------------------------------------------------- | ------- |
| `initialValue` | `boolean \| string[]` | No       | Initial value - either boolean or array of strings to cycle through | `false` |

### Returns

| Name  | Type                | Description                                                 |
| ----- | ------------------- | ----------------------------------------------------------- |
| `[0]` | `boolean \| string` | Current toggle value (boolean or current string from array) |
| `[1]` | `() => void`        | Function to toggle the value                                |

### Features

- Support for boolean toggling
- Support for cycling through string values
- Simple tuple return for destructuring
