# clamp

Constrain a number between a minimum and maximum value.

## Usage

```typescript
import { clamp } from '@libraries/utilities';

clamp(5, 0, 10); // Returns 5
clamp(-5, 0, 10); // Returns 0
clamp(15, 0, 10); // Returns 10
clamp(100, 50, 100); // Returns 100
```

## API Reference

### Import

```typescript
import { clamp } from '@libraries/utilities';
```

### Parameters

| Name    | Type     | Required | Description             | Default |
| ------- | -------- | -------- | ----------------------- | ------- |
| `value` | `number` | Yes      | The number to constrain | N/A     |
| `min`   | `number` | Yes      | Minimum allowed value   | N/A     |
| `max`   | `number` | Yes      | Maximum allowed value   | N/A     |

### Returns

| Name     | Type     | Description                                       |
| -------- | -------- | ------------------------------------------------- |
| (result) | `number` | The constrained number within the specified range |

## Common Use Cases

- Clamping animation progress values (0 to 1)
- Constraining scroll positions
- Limiting volume levels
- Keeping numeric inputs within valid ranges
