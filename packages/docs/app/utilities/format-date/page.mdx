# format_date

Consistently format dates across your application.

## Usage

```typescript
import { format_date } from '@libraries/utilities';

format_date(new Date());
format_date(new Date(), 'YYYY-MM-DD');
format_date(new Date(), 'MM/DD/YYYY');
```

## API Reference

### Parameters

- `date` - Date object or date string
- `format` - Format string (default: 'MMM D, YYYY')

### Format Options

- `YYYY` - 4-digit year
- `MM` - 2-digit month
- `DD` - 2-digit day
- `HH` - 2-digit hour
- `mm` - 2-digit minute
- `ss` - 2-digit second

## Common Formats

```typescript
format_date(new Date()); // "Oct 28, 2025"
format_date(new Date(), 'YYYY-MM-DD'); // "2025-10-28"
format_date(new Date(), 'MM/DD/YYYY'); // "10/28/2025"
```
