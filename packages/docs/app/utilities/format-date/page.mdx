# format_date

Consistently format dates across your application.

## Usage

```typescript
import { format_date } from '@libraries/utilities';

format_date(new Date());
format_date(new Date(), 'short');
format_date(new Date(), 'long');
```

## API Reference

### Import

```typescript
import { format_date } from '@libraries/utilities';
```

### Parameters

| Name     | Type                | Required | Description                                                 | Default   |
| -------- | ------------------- | -------- | ----------------------------------------------------------- | --------- |
| `date`   | `Date \| string`    | Yes      | Date object or valid date string                            | N/A       |
| `format` | `'short' \| 'long'` | No       | Format type: 'short' (DD/MM/YYYY) or 'long' (full readable) | `'short'` |

### Returns

| Name        | Type     | Description               |
| ----------- | -------- | ------------------------- |
| (formatted) | `string` | The formatted date string |

### Format Options

- `'short'` - DD/MM/YYYY format
- `'long'` - Full readable format (e.g., "Monday, January 15, 2024")

## Common Formats

```typescript
format_date(new Date()); // "15/01/2024"
format_date(new Date(), 'short'); // "15/01/2024"
format_date(new Date(), 'long'); // "Monday, January 15, 2024"
```
