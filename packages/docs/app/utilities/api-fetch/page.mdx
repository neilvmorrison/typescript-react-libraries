# api_fetch

Type-safe API requests with comprehensive error handling.

## Usage

```typescript
import { api_fetch, api_get, api_post, api_delete } from '@libraries/utilities';

// Basic fetch
const { data, error } = await api_fetch('/api/users');

// GET request
const { data: users } = await api_get('/api/users');

// POST request
const { data: newUser } = await api_post('/api/users', {
  name: 'John Doe',
  email: 'john@example.com',
});

// DELETE request
await api_delete(`/api/users/${userId}`);
```

### Error Handling

All functions return a `IResult<T, IApiError>` object:

```typescript
interface ISuccess<T> {
  data: T;
  error: null;
}

interface IError<E> {
  data: null;
  error: E;
}

type IResult<T, E = unknown> = ISuccess<T> | IError<E>;
```
