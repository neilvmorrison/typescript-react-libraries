# debounce

Delay function execution until after activity stops for specified duration.

## Usage

```typescript
import { debounce } from '@libraries/utilities';

const handleSearch = debounce((query: string) => {
  console.log('Searching for:', query);
}, 300);

<input onChange={(e) => handleSearch(e.target.value)} />
```

## API Reference

### Import

```typescript
import { debounce } from '@libraries/utilities';
```

### Parameters

| Name    | Type                             | Required | Description           | Default |
| ------- | -------------------------------- | -------- | --------------------- | ------- |
| `fn`    | `T extends (...args) => unknown` | Yes      | Function to debounce  | N/A     |
| `delay` | `number`                         | Yes      | Delay in milliseconds | N/A     |

### Returns

| Name        | Type                               | Description                                                                                                |
| ----------- | ---------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| (debounced) | `(...args: Parameters<T>) => void` | A debounced function that delays execution until after the specified delay has elapsed since the last call |

## Features

- Automatic cleanup
- Prevents excessive function calls
- Useful for search inputs, window resize, etc.

## Common Use Cases

- Search input handling
- Auto-save functionality
- Window resize handlers
- Form field validation
