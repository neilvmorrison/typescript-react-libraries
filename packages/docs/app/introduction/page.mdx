# Introduction

Welcome to **@libraries** - a production-ready collection of React hooks, TypeScript utilities, and UI components.

## What is @libraries?

@libraries is a monorepo containing three carefully crafted packages designed to accelerate your React and TypeScript development:

1. **@libraries/hooks** - Essential React hooks for common use cases
2. **@libraries/utilities** - TypeScript utilities for data manipulation and API communication
3. **@libraries/components** - Accessible and composable React components

Install the packages you need:

```bash

npm install @libraries/hooks @libraries/utilities @libraries/components

# Or install individually
npm install @libraries/hooks
npm install @libraries/utilities
npm install @libraries/components
```
